## How to Run the Project

Frontend Setup
* Navigate to the frontend directory.
* Install all dependencies by running:
  * npm install
* Start the development server:
  * npm run dev
* The application will be available at the URL shown in your terminal (http://localhost:3000).

Backend Setup
* Open the application.properties file located in the backend project.
* Add the required configuration values (e.g., database credentials, AWS S3 bucket details, etc.).
  * ⚠️ Note: Ensure that your S3 bucket has public access enabled for testing purposes.
* Run the backend application
* The backend will start on the configured port (http://localhost:9292).
